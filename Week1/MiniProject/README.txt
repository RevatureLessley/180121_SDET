J&B Bank, Jan 29,2018 by Jawaun Brown
This package contains a program that allows users to create and use their own bank accounts with our bank, J&B Bank.
To get started, type java BankDesk into the command line.
You will then be prompted to either register or sign in.
First time users should register as Accounts if they would like banking.
Administrators should register in order to approve accounts.
This program works by housing the main method inside of a BankDesk user interface.
The BankDesk allows users to signup and register. Once signed in they are taken to their respective menus to see optional tasks they can perform
